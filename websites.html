TiTiTea Website

2021

For my first website, I created a page that represents a tea shop, named TiTiTea. Clients can order tea, rate the place and even apply to open positions. By making this website, I've learned a lot of important skills in web developing. First, I learned how to design a page. Then, I discoverd how to make the website interact with the client. For instance, if the client doesn't fill all the required fields in a form, a message will show up. Finally, I'm able to store the data inputted by the client into a database and show them that their request has been submitted and sent to the back-end.
